Project Description & Goals:

A cross domain recommender system that recommends items to users by utilizing four different domains: movies, video games, books, and music.

Research Question: How do people decide what they like or dislike?


Methodology:

  • Tools and Libraries:
    • Python
    • Pymongo
    • numpy
    • matplotlib
    • MongoDB
  • Item-Item based collaborative filtering

Conclusion:

  • Recommendations seem to be more accurate when preferences of user in question are mainstream.
  • Recommendations can improve with better distribution of ratings.
  • Item-Item CF is more efficient for recommending items where data is of "unusual size."
  • There is still some possibility for serendipity.
